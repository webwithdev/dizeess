# Dizees - Digital Products Platform

A modern, minimalist digital products platform built with Next.js 14 and Tailwind CSS.

## 🎨 Design

- **Primary Color**: Dark Green (#064E3B)
- **Background**: White
- **Text**: Black
- **Style**: Clean, modern, minimal UI with soft shadows and rounded corners

## 🚀 Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## 📁 Project Structure

```
dizees/
├── app/
│   ├── dashboard/          # Analytics dashboard page
│   ├── products/
│   │   └── productivity/   # Product detail page
│   ├── layout.tsx          # Root layout with navbar
│   ├── page.tsx            # Home page
│   └── globals.css         # Global styles
├── components/
│   ├── Navbar.tsx          # Navigation component
│   └── AnalyticsCard.tsx   # Dashboard analytics card
└── tailwind.config.ts      # Tailwind configuration
```

## 🎯 Features

- **Home Page**: Hero section with feature highlights
- **Productivity Page**: Product details and pricing
- **Dashboard**: Analytics and recent activity tracking
- **Responsive Design**: Mobile-friendly layout
- **Modern UI**: Clean, minimal design with smooth transitions

## 🛠️ Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React 18

## 📝 Notes

This is a personal project for selling digital products. No authentication, database, or backend logic included - it's a frontend-only showcase.
